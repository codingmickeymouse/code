1 a) Create a portfolio website for a Profile

Aim

To design and develop a personal portfolio website using HTML and CSS.

Algorithm

  1. Design a layout for the portfolio.
  2. Create HTML structure for profile, projects, contact, etc.
  3. Style with CSS including responsiveness.
  4. Test in various browsers and devices.

Program

      Copied!
      
      <!DOCTYPE html>
<html>
<head>
  <title>My Portfolio</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <header><h1>John Doe</h1></header>
  <section>
    <h2>Projects</h2>
    <p>Project 1: Description...</p>
  </section>
</body>
</html>

Output

A functional and styled personal portfolio website.

1 b) Create a login form using HTML and CSS

Aim

To design a login form with validation using HTML and CSS.

Algorithm

  1. Create an HTML form with fields for username and password.
  2. Add labels and placeholders for better usability.
  3. Style the form using CSS to center it on the page and enhance appearance.
  4. Ensure responsiveness and clarity in layout.

Program

      Copied!
      
      <!DOCTYPE html>
<html>
<head>
  <title>Login Form</title>
  <style>
    body {
      font-family: Arial;
      background: #f4f4f4;
      display: flex;
      justify-content: center;
      align-items: center;
      height: 100vh;
    }
    .login {
      background: white;
      padding: 20px;
      border-radius: 8px;
      box-shadow: 0 0 10px rgba(0,0,0,0.1);
    }
    input[type="text"], input[type="password"] {
      width: 100%;
      padding: 10px;
      margin: 10px 0;
    }
    input[type="submit"] {
      background: #3498db;
      color: white;
      border: none;
      padding: 10px;
      width: 100%;
      cursor: pointer;
    }
  </style>
</head>
<body>
  <div class="login">
    <form>
      <h2>Login</h2>
      <input type="text" placeholder="Username" required>
      <input type="password" placeholder="Password" required>
      <input type="submit" value="Login">
    </form>
  </div>
</body>
</html>

Output

A responsive login form styled with CSS, ready for integration with backend logic.

1 c) Create a responsive website using Bootstrap

Aim

To create a responsive web layout using Bootstrap framework.

Algorithm

  1. Include Bootstrap CSS and JS via CDN in the HTML file.
  2. Use Bootstrap grid system and responsive classes to organize layout.
  3. Add components like Navbar, Cards, and Footer.
  4. Test responsiveness across screen sizes.

Program

      Copied!
      
      <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap Website</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <nav class="navbar navbar-expand-lg navbar-dark bg-dark">
    <div class="container-fluid">
      <a class="navbar-brand" href="#">MySite</a>
    </div>
  </nav>

  <div class="container mt-5">
    <div class="row">
      <div class="col-md-4">
        <div class="card">
          <div class="card-body">
            <h5 class="card-title">Card 1</h5>
            <p class="card-text">Some quick example text.</p>
          </div>
        </div>
      </div>
      <div class="col-md-4">
        <div class="card">
          <div class="card-body">
            <h5 class="card-title">Card 2</h5>
            <p class="card-text">Another example text.</p>
          </div>
        </div>
      </div>
    </div>
  </div>

  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

Output

A responsive layout using Bootstrap components and grid system.
